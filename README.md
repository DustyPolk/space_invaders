# Space Invaders

A classic Space Invaders clone made with Pygame.

## Features
- Classic gameplay: Move, Shoot, Dodge.
- **New:** Pixel Art Visuals.
- **New:** Start Menu and Game Over screens.
- **New:** Score tracking and High Scores.
- **New:** Endless Levels with increasing difficulty.
- **New:** Starfield background.

## Controls
- **Arrow Left / Right**: Move Ship
- **Space**: Shoot / Start Game
- **R**: Restart (on Game Over)
- **Q**: Quit (on Game Over)

## How to Run
1. Ensure you have [uv](https://docs.astral.sh/uv/) installed.
2. Install dependencies and run the game:
   ```bash
   uv run main.py
   ```
